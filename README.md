# 个人云盘系统 (Personal Cloud Drive)

一个集**文件管理、社交互动、虚拟货币（星币）、社区分享**于一体的全功能个人云盘系统，基于 Flask + SQLite 开发，界面美观、部署简单、功能完整。

## ✨ 系统特色

- **一体化生态**：云盘存储 + 好友社交 + 社区广场 + 星币激励闭环

- **现代UI设计**：全局毛玻璃半透明效果，自适应界面

- **轻量无需数据库服务**：内置 SQLite，开箱即用

- **安全可靠**：密码哈希存储、防SQL注入、文件隔离存储

- **免登录体验**：公开文件支持游客浏览、下载

## 🔧 核心功能

### 1. 用户系统

- 注册/登录：用户名3-20位，密码≥6位

- 7天免登录持久化会话

- 个人中心：修改密码、账号注销、容量/星币查询

### 2. 文件管理

- 支持**任意格式**文件上传，单文件最大256MB

- 文件权限：公开/私密切换

- 在线预览：图片、文本文件直接查看

- 容量管理：默认100MB/用户，支持星币兑换扩容

### 3. 星币系统（虚拟货币）

**获取方式**

- 新用户注册：赠送100星币

- 每日签到：+10星币

- 上传文件：+文件大小(MB向下取整)

- 公开文件获赞：+1星币/次，获收藏：+2星币/次

**使用方式**

- 兑换存储容量：1星币 = 0.5MB

- 查看星币收支明细记录

### 4. 社交与社区

- 好友系统：搜索、添加、接受/拒绝好友

- 好友实时聊天：3秒轮询，已读标记

- 社区广场：所有公开文件按热度排序

- 登录用户可点赞、收藏、下载文件

### 5. 管理员后台

- 管理所有用户信息、文件、星币

- 重置用户密码、删除用户

- 调整用户星币数量

- 查看全站文件存储情况

## 🛠 技术栈

|分层|技术|
|---|---|
|后端|Python 3.12 + Flask 3.1|
|数据库|SQLite3（轻量级、无额外服务）|
|前端|原生 HTML5 + CSS3 + JavaScript|
|UI|Font Awesome 6 图标库|
|安全|Werkzeug 密码哈希、Session 管理、防SQL注入|
|存储|本地文件系统，按用户ID目录隔离|
## 📂 项目结构

```Plain Text

cloud_drive/
├── app.py               # Flask 主程序，所有路由与业务逻辑
├── users.db             # SQLite 数据库（自动生成）
├── uploads/             # 文件存储目录，按用户ID分文件夹
├── static/
│   └── background.png   # 毛玻璃背景图
└── templates/           # 前端页面模板
    ├── login.html
    ├── user_center.html
    ├── community.html
    ├── admin.html
    ├── privacy_policy.html
    └── user_agreement.html
```

## 🚀 快速部署

### 1. 环境要求

Python 3.7 及以上版本

### 2. 安装依赖

```Bash

pip install flask
```

### 3. 启动项目

```Bash

python app.py
```

### 4. 访问系统

浏览器打开：

```Plain Text

http://127.0.0.1:8000
```

## 📖 使用说明

1. **注册/登录**：使用用户名密码注册，支持7天免登录

2. **文件上传**：进入个人中心上传文件，自动获得星币奖励

3. **文件管理**：设置公开/私密，在线预览、删除文件

4. **社区互动**：浏览公开文件，点赞收藏获得/赠送星币

## ⚠️ 注意事项

1. **生产环境必须修改**：`SECRET_KEY` 和管理员默认密码

2. 内置 Werkzeug 服务器仅用于**测试**，生产环境请使用 Gunicorn / Waitress

3. 文件大小限制可在 `app.config["MAX_CONTENT_LENGTH"]` 中调整

4. 建议定期备份 `users.db` 和 `uploads/` 目录

5. 可定期清理无效/孤立文件
